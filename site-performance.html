Site Performance

This was the "last" project in Treehouse TechDegree (the last being this portfolio site). In this project we were once again tasked with refactoring.

We were given a website with a huge load time and dozens of http requests. For optimal performance, websites should make as few requests as possible and load the bare minimum in order to display a good experience to users.

In this website's case, there were many extraneous http requests and all of the images were sized well beyond what they were displaying at. After optimizing the images, and removing unnecessary requests, the page now loads very fast!
